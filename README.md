# goit-advancedjs-hw-03

**Module: Advanced JavaScript and TypeScript**

## Description

Fetching and rendering data from an external API — an image search app powered by Pixabay with lightbox support.

## Tasks

- **Image Search** — sends a GET request to the Pixabay API on form submit, renders a responsive photo gallery, and shows a loader during the request.
- Clicking any image opens it in a SimpleLightbox overlay.
- Error and empty-result states are handled with iziToast notifications.

## Stack

- Vanilla JavaScript (ES6+)
- Vite
- [Axios](https://axios-http.com/)
- [SimpleLightbox](https://simplelightbox.com/)
- [iziToast](https://izitoast.marcelodolza.com/)
